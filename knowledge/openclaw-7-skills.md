# 7 OpenClaw Skills That Run My Business

**Source:** Riley Brown (YouTube)  
**URL:** https://www.youtube.com/watch?v=ryhzpLe9O_U

---

## The 7 Skills

### 1. Notion Skill
- Full control over Notion workspace
- Create docs, read pages, manage databases
- Agent creates Notion docs with links automatically returned
- Example: "Create a notion doc for this script" → instant link

### 2. Transcript Extractor (Any Video/Platform)
- Uses Super Data API to extract transcripts from YouTube, Instagram, Twitter
- Pulls transcripts → saves to Notion for future reference
- Example: "Extract scripts from Kane Callaway's top 5 videos" → creates Notion doc with all transcripts
- Then can write content "in his style" using context

### 3. Twitter / Typefully Skill
- Connects to Typefully API
- Auto-drafts tweets based on content
- Generates multiple caption options
- Can post directly or save as drafts
- Example: Create video → analyzes thumbnail → generates 5 tweet options → drafts in Typefully

### 4. Linear Skill
- Project management integration
- Read/write access to tasks
- Ask: "What tasks do I have?" "What should I prioritize?"
- Creates content calendars from Linear tasks
- Only read access recommended initially

### 5. Nano Banana (Thumbnail Mood Board)
- Uses AI image generation to replace faces in thumbnails
- Pulls top 40 YouTube thumbnails on a topic
- Replaces creator's face with yours
- Creates PDF mood board for thumbnail ideas

### 6. Google Images Skill
- Uses Serper API to search Google Images
- Automatically adds relevant images to Notion docs
- Example: Script about "Meta acquiring OpenClaw" → adds relevant images

### 7. "Take the Wheel" Skill (Favorite)
- Flips agent to urgent mode
- Asks ONE critical question at a time (in ALL CAPS)
- Creates sense of urgency, pushes task to completion
- Example: "Create 20 ad scripts" → agent enters urgent flow state, asks questions until done
- Perfect for: long-form video scripts, ad batches

---

## Key Insight

Skills are just markdown files. You tell the agent: "Create a skill that does X" → it builds the skill file itself.

## Related Notes

- Riley runs 15 accounts with 2.5M followers using OpenClaw
- Uses Mac Mini for local deployment
- Connects via Telegram (recommended over WhatsApp/iMessage)
- Skills can be shared between agents via shared Notion notebook
- Warning: Don't create agents with too many skills - becomes chaotic. Keep focused.
