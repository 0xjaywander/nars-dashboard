# Research Quality Assessment Criteria

## Readiness Checklist

### Completeness
- [ ] All sub-topics from task covered
- [ ] Key players/companies identified
- [ ] Market data included (sizes, growth rates)
- [ ] Risks and challenges addressed
- [ ] Sources cited for all claims

### Depth
- [ ] More than surface-level summary
- [ ] Actionable insights provided
- [ ] Nuances explained
- [ ] Trade-offs identified

### Structure
- [ ] Clear title and summary
- [ ] Logical section organization
- [ ] Consistent formatting
- [ ] Easy to navigate

### Accuracy
- [ ] Facts distinguished from opinions
- [ ] Claims verified with sources
- [ ] Dates and numbers cross-checked
- [ ] No obvious gaps in logic

## Scoring Guide

| Score | Meaning | Action |
|-------|---------|--------|
| 95-100 | Excellent | Ready to share |
| 75-94 | Good with minor gaps | Iterate on gaps |
| 60-74 | Partial coverage | Significant revision |
| <60 | Not ready | Major rework |

## Iteration Protocol

```
Draft Produced
     ↓
Main Agent Evaluation
     ↓
┌────┴────┐
↓         ↓
<95%      ≥95%
↓         ↓
Feedback  Ready
→ Issues  → Share
→ Next    → Archive
 Draft
```
